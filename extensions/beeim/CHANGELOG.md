# Changelog

## [1.0.0] - 2026-03-30

### Added

- Initial release of `openclaw-beeim` channel plugin
- Based on `openclaw-nim` v1.0.1, adapted for BeeIM branding
- Full NIM (NetEase IM) SDK integration via `@yxim/nim-bot`
- Multi-instance support (up to 3 instances)
- P2P (private chat) and Team (group chat) message support
- QChat (圈组) support
- Media message support: image, file, audio, video
- Streaming message support
- Access control: open / allowlist / disabled policies
- Private deployment configuration support
